# To-Do list
"To-do list" is a tool that helps to organize your day. It simply lists the things that you need to do and allows you to mark them as complete.

![screenshot](./app.png)

## Built With

- Javascript
- HTML 5
- CSS 3
- React

## Usage
### Local installation
```console
$ git clone git@github.com:vicmaburrito/to-do_list_react.git
$ cd to-do_list_react
$ npm install
```

### To run the project
```console
$ npm start 
```
